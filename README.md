### Instalar npm

```
npm install
```

### Run

```
npm run start
```

### Formato JSON

```
Datos que acepta el formato JSON al hacer peticiones en POSTMAN:
{
    "typeID": (number),
    "photo": "String",
    "phone": "String",
    "email": "String",
    "name": "String"
}
```
